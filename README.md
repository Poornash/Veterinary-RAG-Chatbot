## Overview
PawMedBot is an offline AI-powered veterinary assistant built using
Retrieval-Augmented Generation (RAG). It helps pet owners by answering
questions related to pet health, diseases, nutrition, and care.

## Tech Stack
- Python 3.11
- Streamlit
- LangChain (modular)
- ChromaDB
- HuggingFace Embeddings
- Ollama (Phi-3 mini)
- SQLite
